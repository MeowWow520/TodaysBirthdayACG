## Today's Birthday in ACG

### 由来

某人 ~~闲的没事干~~ 突发奇想记录自己认识的 ACG 角色的生日

其实是注册 x(原 Twitter) 后关注了最近看到番剧之后，偶然收到了官方发的角色生日祝福恰好是当时的头像

### 记录形式

- 分类
在 `Birthdays` 文件夹中以“月份”为类别，使用 `day-xx.xx.md` 记录。
如 `./Birthdays/day-01.20.md`

- 记录
在 `.md` 文件中，按照以下的主框架：
```md
#### 时间
##### 角色 - 1
**其他名字**： | 英文名 | 日文名 |
(可选)**<角色>的别称**：角色的其他名字
**来自<形式>**：番剧/漫画/小说的名字
(可选)**<番剧/漫画/小说名字>的别称**：番剧/漫画/小说的其他名字
**萌属性**：如：发色、性格
```
